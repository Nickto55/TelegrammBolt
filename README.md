## Bolt - Система связи преимущественно через телеграмм бота и сайт
## Системные требования
```
-  **ОС**: Debian 10+ или Ubuntu 18.04+
-  **Python**: 3.8+
-  **Память**: минимум 512 MB RAM
-  **Место**: ~200 MB свободного места
```



## Способы установки:
**Установка проекта**
1. Клонируйте репозиторий
   ```
   git clone <repo-url>
   cd TelegrammBolt
   ```
2. Запустите скрипт установки
   ```
   chmod +x setup.sh./setup.sh
   ./setup.sh
   ```
3. Используйте панель управления
   ```
   chmod +x manage.sh- Обновит систему
   ./manage.sh
   ```



**Для установки необходимо:**
1. Бот токен из телеграмма
   *получите от [@BotFather](https://t.me/BotFather) в Telegram*
2. ID пользователя из телеграмма.
   *ваш Telegram ID (узнайте у [@userinfobot](https://t.me/userinfobot))* -  необходимо для выдачи админ прав
3. Домен и SSL сертификат (опционально)
   *Можно не подключать или использовать самоподписный сертификат*
4. SMTP сервер



## Способы изменения портов
1. **Через config/domain.conf:**
   ```
   properties
   WEB_PORT=8080
   ``` 
2. **Через переменную окружения:**
   ```
   bashchmod +x manage.sh
   WEB_PORT=8080 ./manage.sh./manage.sh
   ```

├── bot/                      # Telegram бот

│   ├── commands.py              # Команды бота
│   └── ...

│   ├── web_app.py               # Flask приложение
